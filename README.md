# QABot-Project
Proyecto QA Bot para análisis de documentos PDF: migrado de arquitectura propietaria en Watsonx a stack 100% open source con LangChain, ChromaDB y Llama 3.3 vía Groq para inferencia ultra‑rápida. Incluye interfaz interactiva en Gradio, optimizando recuperación y generación de respuestas.
